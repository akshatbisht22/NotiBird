# **Notibird**

<strong>Overview :</strong>
Notibird is a secure place to create and keep your memories safely. A note app built with PHP , Javascript and MySQL.
<br>

<strong>Features :</strong>
- Simple UI design
- Fast and Secure Servers
- A huge Userbase
